# EVAlger-无人机电力线路巡检数据分析软件

#### Description
本软件是根据《架空输电线路多旋翼无人机巡检影像拍摄指导手册》（国家电网有限公司2019年6月V2版）设计，同时参考了《缺陷等级划分》内容，实现了无人机电力线路巡检照片数据处理过程一体化、自动化，操作简单、方便。同时可一步完成响应图像文件命名转存、报告输出等，为数据处理标准化提供了强有力的工具。
